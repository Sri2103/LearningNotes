# DSA Skill Map

## Fundamentals

- Time & space complexity
- Big-O intuition
- Arrays, strings, hashmaps
- Stacks & queues
- Recursion basics

## Intermediate

- Sliding Window
- Two Pointers
- Prefix Sum
- Binary Search
- BFS & DFS
- Backtracking
- Monotonic stack

## Advanced

- Dynamic Programming (1D/2D)
- Graph algorithms (Dijkstra, Topological Sort)
- Union-Find
- Tries
- Heaps / priority queues
