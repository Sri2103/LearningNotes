# Algorithms & Math Trigger Checklist

- Two nested loops → consider O(n log n) or O(n)
- Long string → think prefix functions (KMP, Z)
- Weighted graph → Dijkstra or Bellman-Ford
- Many queries → prefix sums or segment tree
- Min/Max → heap
- Subarrays → sliding window
- Multiple components → union-find
- Counting paths → DP
- Modular operations → use mod + fast exponentiation
