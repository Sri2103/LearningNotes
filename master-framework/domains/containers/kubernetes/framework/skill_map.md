# Kubernetes Skill Map

## Fundamentals

- Pods
- Deployments / ReplicaSets
- Services (ClusterIP, NodePort)
- Namespaces
- Labels & selectors
- ConfigMaps
- Secrets

## Intermediate

- Liveness & Readiness probes
- Ingress
- Volumes & Persistent Volumes
- Autoscaling (HPA)
- DaemonSets
- StatefulSets
- NetworkPolicies

## Advanced

- CRDs
- Controllers & Operators
- Mutating/Validating Webhooks
- Controller-runtime / Kubebuilder
- Cluster architecture
- HA control planes
- Security (RBAC, PSP)
- Service Mesh (Istio)
