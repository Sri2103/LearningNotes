# DevOps/SRE Mistake Log

## Common Mistakes I've Made

- Editing infrastructure manually
- Forgetting to set resource limits
- Not checking logs during incidents
- Misconfigured alerts
- Running commands on prod directly
- No testing in CI/CD
- Ignoring costs

## Fix Strategy

- Everything as code
- Always write tests
- Validate before deploy
- Follow incident runbooks
- Set budgets and quotas
