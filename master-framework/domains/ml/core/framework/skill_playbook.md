# ML/DL Playbook

## Modeling Playbook

1. Define objective
2. Understand data distribution
3. Pick baseline model
4. Choose evaluation metrics
5. Train → validate → iterate
6. Monitor overfitting
7. Compare models
8. Deploy best one

---

## Deep Learning Architecture Playbook

- CNN → image tasks
- RNN/LSTM → sequence/time-series
- Transformers → everything (NLP, vision)
- Autoencoders → denoising, latent representations
- GANs → generative tasks

---

## Optimization Playbook

- start with Adam
- reduce LR on plateau
- monitor gradients
- use dropout, batchnorm
- early stopping

---

## Data Playbook

- normalize/standardize
- split data carefully
- augment training data
- remove leakage
- fix imbalanced datasets

---

## MLOps Playbook

- version datasets
- version models
- build CI for training
- serve via API or batch jobs
- add drift detection
- automate retraining
