# Networking Skill Map

## Fundamentals

- OSI model
- TCP/IP
- IP addressing
- Subnetting
- Routing basics
- Switching (L2)
- ARP
- DNS

## Intermediate

- TCP handshake
- TCP flags
- Flow control (rwnd)
- Congestion control (cwnd)
- UDP
- DHCP
- NAT
- HTTP/1.1, HTTP/2
- TLS handshake

## Advanced

- Load balancing (L4/L7)
- Reverse proxies
- CDN internals
- QUIC & HTTP/3
- Deep packet inspection
- Sockets & kernel network stack
- VPNs, tunneling
- BGP & autonomous systems
