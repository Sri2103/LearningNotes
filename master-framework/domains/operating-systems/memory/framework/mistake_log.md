# Memory & Storage Mistake Log

## Common Mistakes I've Made

- Mixing up physical vs virtual memory
- Thinking swapping = memory leak
- Misunderstanding cache eviction
- Using large structs without alignment
- Ignoring NUMA architecture
- Confusing SSD latency with RAM latency

## Fix Strategy

- Use diagrams
- Use benchmarking tools
- Revisit CPU/memory hierarchy weekly
- Use perf, cachegrind
